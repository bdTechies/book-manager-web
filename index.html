Book Manager

A simple crossplatfotm desktop app to manage personal library.

  • Windows
  • Mac
  • Linux
Download

Features

Add Book Info
You can easily add information like title, author, publisher, reading status etc.
Search Book
You can easily search book from the stored data by title, author and publisher.
Take Notes
While reading books you can easily take notes with in app WYSIWYG editor.
Export & Import Data
You can easily export saved data in json form and import it againg for backup.

Book Manager is not a book reader. It does not provide any way of storing any kind book files. It only helps you to manager your little library and take notes.
