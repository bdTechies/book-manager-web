Features

Add Book Info
You can easily add information like title, author, publisher, reading status etc.
Search Book
You can search book from the stored data by title, author and publisher.
Take Notes
While reading books you can easily take notes with an WYSIWYG editor.
Export & Import Data
You can easily export saved data in json form and import it againg for backup.
